# ClawCompany Demo 脚本

## 🎬 Demo 概述

**产品名称：** AI Team - 一人公司 AI 团队
**目标用户：** 独立开发者、创业者、自由职业者
**核心价值：** 让一个人也能拥有完整的 AI 团队（PM + Dev + Review）

**Demo 时长：** 2-3 分钟
**录制工具：** macOS 屏幕录制 / OBS
**分辨率：** 1920x1080

---

## 📝 Demo 流程

### 第一幕：Landing Page（15秒）

**画面：** 浏览器打开 `http://localhost:3000`

**配音/字幕：**
> "一人公司，无限可能。AI Team 让你一个人也能拥有完整的 AI 团队。"

**展示内容：**
- ✨ Landing Page 动画效果
- 📋 三个 Agent 卡片（PM、Dev、Review）
- 🎯 "Start Chatting" 按钮

**操作：**
1. 打开浏览器
2. 访问 `http://localhost:3000`
3. 等待动画完成
4. 鼠标悬停在 Agent 卡片上

---

### 第二幕：Team Portal（10秒）

**画面：** 点击 "Start Chatting" → 进入 Team Portal

**配音/字幕：**
> "只需要一个需求，AI 团队就会自动协作，帮你完成整个项目。"

**展示内容：**
- 💬 聊天界面
- 🤖 三个 Agent 头像
- ⌨️ 输入框

**操作：**
1. 点击 "Start Chatting" 按钮
2. 等待页面加载
3. 展示聊天界面

---

### 第三幕：完整协作流程（60秒）

**画面：** 输入需求 → PM Agent 分析 → Dev Agent 开发 → Review Agent 审核

**配音/字幕：**
> "让我创建一个登录页面..."
> 
> **PM Agent：** "好的，我来分析需求..."
> 
> **Dev Agent：** "我来实现这个功能..."
> 
> **Review Agent：** "我来审核代码质量..."

**展示内容：**
- 💬 用户输入需求
- 🤖 PM Agent 响应（分析需求）
- 💻 Dev Agent 响应（生成代码）
- ✅ Review Agent 响应（审核代码）

**操作：**
1. 在输入框输入："帮我创建一个登录页面"
2. 点击发送按钮
3. 等待 PM Agent 响应（约 10 秒）
4. 等待 Dev Agent 响应（约 10 秒）
5. 等待 Review Agent 响应（约 10 秒）
6. 展示完整对话历史

---

### 第四幕：成果展示（15秒）

**画面：** 展示完整对话历史和代码

**配音/字幕：**
> "从需求到代码，只需要 1 分钟。AI Team 让你专注于创意，而不是重复劳动。"

**展示内容：**
- 📋 完整对话历史
- 💻 生成的代码
- ✅ 审核结果

**操作：**
1. 滚动查看完整对话
2. 高亮显示关键代码片段
3. 展示 Review Agent 的审核结果

---

### 第五幕：价值主张（10秒）

**画面：** 回到 Landing Page 或展示 Logo

**配音/字幕：**
> "一人公司，无限可能。立即开始你的 AI 团队之旅。"

**展示内容：**
- 🎯 产品 Logo
- 📧 联系方式
- 🔗 GitHub 链接

## 🎯 Demo 录制最佳实践

### 录制前

1. **测试流程：** 先跑一遍完整流程，确保没有错误
2. **准备数据：** 准备好测试需求（"帮我创建一个登录页面"）
3. **环境检查：** 确保网络稳定，API 正常

### 录制中

1. **保持流畅：** 避免鼠标移动过快或过慢
2. **留出时间：** 等待响应时，鼠标停留在输入框附近
3. **避免干扰：** 关闭通知、消息提醒

### 录制后

1. **检查质量：** 播放一遍，确保清晰度和流畅度
2. **剪辑优化：** 删除不必要的等待时间
3. **添加字幕：** 帮助观众理解内容
