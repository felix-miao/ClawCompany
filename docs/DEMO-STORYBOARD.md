## 📝 详细操作步骤

### 镜头 1-3（20秒）
```
1. 打开浏览器，访问 http://localhost:3000
2. 等待页面加载完成（5秒）
3. 慢慢滚动，展示三个 Agent 卡片（10秒）
4. 点击 "Start Chatting" 按钮（5秒）
```

### 镜头 4-9（60秒）
```
5. 等待 Chat Page 加载（5秒）
6. 在输入框输入："创建一个登录页面"（10秒）
7. 点击 "Send"，等待响应（15秒）
8. 右侧任务列表出现，鼠标指向任务（10秒）
9. Dev Agent 和 Review Agent 消息出现（15秒）
10. 慢慢滚动，展示完整对话（10秒）
11. 点击左上角 "← Back"（5秒）
```

### 镜头 10-13（35秒）
```
12. 回到首页，点击 "View Demo"（5秒）
13. Demo Page 加载，观看打字动画（15秒）
14. 等待几轮对话完成（10秒）
15. 停留在 Demo Page（5秒）
```

### 镜头 14-17（30秒）
```
16. 新标签页打开 GitHub：https://github.com/felix-miao/ClawCompany
17. 滚动 README，展示项目介绍（10秒）
18. 点击 docs/ARCHITECTURE-v2.md，展示架构图（10秒）
19. 切回 Landing Page，结束（5秒）
```

## 💡 演示技巧

### 画面控制
1. **慢** - 所有操作都要慢
2. **稳** - 鼠标不要抖
3. **停** - 关键画面停 3 秒
4. **滑** - 滚动要平滑

### 节奏控制
- 前 20 秒：吸引注意力（炫酷动画）
- 中 60 秒：展示核心功能（真实交互）
- 后 35 秒：展示可用性（GitHub）
- 最后 15 秒：留下印象（回到首页）
