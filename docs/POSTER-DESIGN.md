## 设计风格

### 配色方案
- **主色：** 蓝色渐变 (#3B82F6 → #8B5CF6)
- **辅助色：** 紫色、绿色、橙色（用于 Agent 卡片）
- **背景：** 深色系 (#0F172A)
- **文字：** 白色、灰色

### 字体
- **标题：** Montserrat Bold / 思源黑体 Bold
- **正文：** Inter Regular / 思源黑体 Regular
- **代码：** JetBrains Mono

### 视觉元素
- 渐变背景
- 毛玻璃效果
- 圆角卡片
- 阴影效果
- 粒子动画（如果支持）

## 快速制作步骤（使用 Canva）

### 步骤 1：选择模板
1. 打开 Canva
2. 搜索 "科技海报" 或 "项目海报"
3. 选择一个简洁的模板

### 步骤 2：替换内容
1. 替换标题为 "ClawCompany"
2. 添加副标题和核心价值
3. 插入架构图（可以简单手绘）
4. 添加使用场景
5. 插入二维码

### 步骤 3：调整样式
1. 修改配色为蓝色渐变
2. 调整字体为现代风格
3. 添加圆角卡片效果
4. 调整间距和对齐

### 步骤 4：导出
1. 导出为 PDF（打印用）
2. 导出为 PNG（网络用）
3. 分辨率：300 DPI
