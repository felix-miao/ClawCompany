## 🎬 Demo 演示流程

### 1. Landing Page（30秒）

**访问：** http://localhost:3000

**展示内容：**
- ✨ 炫酷的 Hero 动画
- 📋 PM Agent 卡片
- 💻 Dev Agent 卡片
- 🔍 Review Agent 卡片
- 🎯 "One Person. Infinite Power." 标语

**旁白：**
> "ClawCompany - 一人企业家，无限可能。通过 AI 虚拟团队，一个人就能像拥有一支完整团队一样工作。"

---

### 2. Demo Page（1分钟）

**点击：** "View Demo" 按钮

**访问：** http://localhost:3000/demo

**展示内容：**
- 模拟的 Agent 对话
- PM Agent 拆分任务
- Dev Agent 实现代码
- Review Agent 审查质量

**旁白：**
> "看，这是我们的 AI 团队协作演示。PM Agent 分析需求，Dev Agent 写代码，Review Agent 审查质量。"

---

### 3. Chat Page（1分钟）

**点击：** "Start Chatting" 按钮

**访问：** http://localhost:3000/chat

**演示操作：**
1. 输入： "帮我创建一个登录页面"
2. 等待 PM Agent 响应
3. 查看任务列表
4. 查看 Agent 对话

**旁白：**
> "现在让我们试试真实的交互。发送一个需求，AI 团队会自动协作完成。"

---

### 4. 架构展示（30秒）

**展示：** GitHub 仓库

**访问：** https://github.com/felix-miao/ClawCompany

**展示内容：**
- README.md
- docs/ARCHITECTURE-v2.md
- 代码结构

**旁白：**
> "我们的架构基于 OpenClaw。OpenClaw 作为'包工头'，spawn 多个 AI agent 组建虚拟团队。"
