## 🎬 录制流程（总时长：2分30秒）

### Part 1: Landing Page（30秒）

**操作：**
1. 打开 http://localhost:3000
2. 等待页面加载（炫酷动画）
3. 慢慢滚动，展示三个 Agent 卡片
4. 停顿 3 秒

**旁白（可选）：**
> "ClawCompany - 一人企业家，无限可能。
> 通过 PM、Dev、Review 三个 AI Agent，
> 一个人也能像拥有一支完整团队一样工作。"

---

### Part 2: Chat Page（90秒）⭐ 重点

**操作：**
1. 点击 "Start Chatting"
2. 等待页面加载
3. 输入："帮我创建一个计算器应用"
4. 点击 Send
5. 等待 PM Agent 响应（5-10秒）
6. **强调：看，这是真实的 GLM-5 智能分析！**
7. 右侧任务列表出现
8. Dev Agent 和 Review Agent 消息出现
9. 慢慢滚动展示完整对话

**旁白（可选）：**
> "发送需求后，PM Agent 会智能分析并拆分任务。
> 注意，这是真实的 GLM-5 调用，不是硬编码的回复。
> Dev Agent 会生成代码，Review Agent 会审查质量。
> 整个过程完全自动化。"

---

### Part 3: Demo Page（30秒）

**操作：**
1. 点击左上角 "← Back"
2. 点击 "View Demo"
3. 观看打字动画
4. 等待几轮对话完成

**旁白（可选）：**
> "这是模拟的演示模式，展示 Agent 协作的动画效果。"

---

### Part 4: GitHub（30秒）

**操作：**
1. 新标签页打开 GitHub
2. https://github.com/felix-miao/ClawCompany
3. 滚动 README
4. 点击 `skill/` 目录
5. 展示 SKILL.md
6. 展示代码实现

**旁白（可选）：**
> "项目完全开源，可以作为 ClawHub plugin 安装使用。
> 核心代码使用 OpenClaw 的 sessions_spawn API，
> 真实调度 PM/Dev/Review 三个 Agent 协作。"
